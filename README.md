# Comparación de Soluciones Analíticas y Numéricas

## Descripción
Este proyecto compara la solución exacta de una ecuación diferencial separable con una aproximación numérica usando el método de Euler.

## Ecuación diferencial
dy/dt = -y  
Condición inicial: y(0) = 1

## Solución analítica
Usando separación de variables se obtiene:
y(t) = e^(-t)

## Método numérico
Se aplica el método de Euler en el intervalo t ∈ [0,1] con paso h = 0.2.

## Resultados
La solución analítica muestra el comportamiento exacto del sistema, mientras que el método de Euler aproxima la solución paso a paso y presenta error numérico.
